四、V2.0 修复的核心问题 (已解决)
──────────────────────────────────────────────────────────────────────────────
根因: 板子 Buildroot 没装 ffmpeg!
修复: camera_manager.py 重写为持久 v4l2-ctl 流式采集
  - v4l2-ctl --stream-mmap --stream-to=/dev/stdout → Python pipe → MJPEG解析
  - 采集性能: 0fps(失败) → 40fps+ (5秒228帧)
  - 新增 get_latest_jpeg() 供 HTTP 接口使用

其他修复:
  - http_server.py: /camera/frame 改用 get_latest_jpeg()
  - main.py: SKIP_BLE/SKIP_HUD 开关, ble.stop()空指针修复
  - camera_config.py: 注释更新 (ffmpeg→v4l2-ctl)
