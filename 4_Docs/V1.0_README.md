## 四、板子部署步骤

```bash
# 1. 上传代码
scp -r board/* root@<板子IP>:/opt/hud_navi/

# 2. 初始化WiFi (第一次)
ssh root@<板子IP>
cd /opt/hud_navi && python wifi_setup.py
# → 手机连HUD-Setup热点 → 浏览器打开192.168.4.1 → 输WiFi密码

# 3. 启动
cd /opt/hud_navi && python main.py

# 4. 开机自启 (已配置)
# /etc/rc.local → /etc/init.d/S99hudnavi → 等WiFi → 启动watchdog → main.py
```

## 六、API接口

```
GET  /ping              → {"status":"alive"}
GET  /status            → 完整导航状态JSON
POST /destination       ← {"lat":xx,"lon":xx,"name":"xx"}  → 开始导航
GET  /cancel            → 取消导航
GET  /places            → 常用地点列表
GET  /stats             → 骑行统计数据
POST /voice/enroll      → 录入声纹
POST /voice/cmd_enroll  → 录入语音命令
GET  /ai/start          → 启动AI检测
GET  /restart           → 重启板子
GET  /shutdown          → 关机
```
