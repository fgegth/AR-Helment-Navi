🟢 适合 640×400 的 UI 元素
  • 大号方向箭头 (80-120px)
  • 转弯距离数字 (30-40px 字体)
  • 路名/街道名 (16-20px 字体)
  • 速度/ETA 小字信息栏
  • 电池/GPS 状态小图标
  • 简洁的路线线框图 (非详细地图)
🔴 不适合的元素
  • 高德静态地图底图 (看不清)
  • 多色路线 + 密集标记
  • 多行小字信息栏
  • 复杂图标/Logo
  • 侧边栏/多面板布局
  • 任何 < 12px 的文字

完整硬件清单

🧠 主控 — QSM368ZP-WF

SoC: RK3568, 4×A55@2.0GHz
GPU: Mali G52
NPU: 1 TOPS
RAM: 2GB LPDDR4X
OS: Linux Kernel 4.19
供电: DC 10-12V

📡 GPS — ATGM336H-5N-31

芯片: 中科微 GPS+北斗双模
接口: TTL UART 9600bps
协议: NMEA 0183
激活: 无需AT, 上电自动
接法: TX→主控RX, RX→主控TX

🖥 屏幕 — SONY ECX336C

类型: 0.23" Micro OLED
分辨率: 640×400
亮度: 3000 cd/m²
驱动板: HDMI, 3.3-5V, 0.3W
光学: 24°FOV, 3m=50寸

🔋 电源 — 3S 锂电池组

构成: 3S LiPo / 3×18650
电压: 9.0V ~ 12.6V
监测: ADC2, 分压 ~7.67:1
降压: DC-DC 12V→5V(屏幕)
续航: 约 5-6 小时
